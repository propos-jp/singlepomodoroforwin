Single pomodoro for Winodws
====

概要

25分-5分のサイクルを繰り返すシンプルなPomodoro Timerです。

## デモ

 ![DEMO](https://raw.github.com/wiki/propos-jp/singlepomodoroforwin/images/pomodoro-demo.gif)


## 動作環境

- Windows7以上の環境で動作します。
- .NET 4.0 Frameworkが必要です
-- [Microsoft .NET Framework 4 (Web インストーラー)](http://www.microsoft.com/ja-jp/download/details.aspx?id=17851)より入手してください
